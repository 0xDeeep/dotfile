# dotfiles

My personal terminal setup for macOS (Apple Silicon).

## Stack

- **Terminal:** Alacritty
- **Multiplexer:** Zellij
- **Shell:** zsh + starship + fzf + autosuggestions + syntax-highlighting
- **Renderer:** glow (markdown)

## Install

```zsh
git clone git@github.com:0xDeeep/dotfile.git ~/dotfiles
cd ~/dotfiles && ./install.sh
```

> Trust but verify. Use at your own risk.
